1) What is the difference between null and undefined?
A: Null is used intentionally to represent "no value".Undefined happens automatically when a variable is declared but not assigned a value.
2) What is the use of the map() function in JavaScript? How is it different from forEach()?
A: Map() creates a new array by applying a function to every element of the original array.ForEach() just runs a function on each element, but does not return a new array.
3) What is the difference between == and ===?
A: == compares only the value (type is ignored).=== compares both value and type.
4) What is the significance of async/await in fetching API data?
A: Using async/await lets you write asynchronous code in a synchronous style.It makes handling promises easier and the code easier to read.
5) Explain the concept of Scope in JavaScript (Global, Function, Block).
A: Global Scope: Accessible from anywhere in the code.Function Scope: Accessible only inside the function.Block Scope: Accessible only inside {} (for let and const).
